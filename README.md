# EURUSDProphet

Mathematical analysis and forecasting models for EUR/USD exchange rate prediction using time series analysis and statistical methods.

## Overview

This project applies mathematical modeling techniques to analyze and predict EUR/USD currency pair movements. The analysis includes statistical testing, trend analysis, and quantitative forecasting models.

## Project Structure

```
├── data/              # EUR/USD historical data
├── models/            # Model implementations and analysis
├── config/            # Configuration settings
├── sample.ipynb       # Main analysis notebook
└── requirements.txt   # Dependencies
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Open the Jupyter notebook to explore the analysis:

```bash
jupyter notebook sample.ipynb
```

## Dependencies

- numpy - Numerical computing
- pandas - Data manipulation
- sympy - Symbolic mathematics
- matplotlib - Visualization
